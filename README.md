# XO_Game

## Game Help :

1. Two player game.
2. Each player will get 4 chance to play a game.
3. Any one either 'X'  or 'O' can start the Game.
4. Drag and Drop 'X' or 'O' on the game Board.


## How to Start: 
1. Run index.html
2. Start Drag and Drop
